# Spectral Lifter Plus

AI 生成音源などで起きやすい「高域の物足りなさ」と「シュワシュワしたノイズ感」を補正し、そのあと独立したマスタリング機能で仕上げまで行う macOS ネイティブアプリです。

## 今の構成
- 画面: `SwiftUI`
- 音声読み書き: `AVFoundation`
- 周波数処理: `Accelerate`
- 実行方法: `./script/build_and_run.sh`

Python や Gradio は使っておりません。  
音声処理はすべて Swift 側で完結します。

## 使い方
1. `./script/build_and_run.sh` を実行します
2. アプリで音声ファイルを選びます
3. `補正を実行` を押します
4. 必要なら `自然 / 配信向け / 前に出す` から仕上がりを選びます
5. `マスタリングを実行` を押します
6. 入力ファイルと同じ場所に `*_lifter` と `*_lifter_mastered` が保存されます

## 処理の流れ
1. 音声読み込みと 48kHz への調整
2. 補正段でスペクトル解析、ノイズ除去、高域補完、帯域調整
3. `*_lifter` を書き出し
4. 補正済み音源を再解析
5. マスタリング段でトーン、帯域バランス、密度、広がり、音量を調整
6. `*_lifter_mastered` を書き出し

## テスト
- `swift build`
- `swift test`

## 注意
- 音質処理はヒューリスティック中心です
- 元音源により効き方は変わります
- 長尺ファイルでは処理時間が伸びます
- マスタリングは補正後音源を前提にしています
