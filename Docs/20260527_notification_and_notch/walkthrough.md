# 変更内容報告書 (Walkthrough)

## 1. 概要
補正またはマスタリングが完了した時に、macOS標準の音付きローカル通知で知らせます。

## 2. 実施された作業内容
- `NotificationService.swift` を追加し、通知許可申請と音付きローカル通知をまとめました。
- `VelouraLucentApp.swift` で、アプリ起動時に通知許可を求めるようにしました。
- `ProcessingJob.swift` で、補正完了とマスタリング完了の通知を各処理につき1回だけ送るようにしました。
- 設定画面で「完了通知」をOFFにすると、補正完了とマスタリング完了の通知を送らないようにしました。
- 通知登録に失敗した時はログにエラー内容を残すようにしました。

## 3. テストと動作検証結果
- 補正完了通知が1回だけ記録されることを単体テストで確認します。
- マスタリング完了通知が1回だけ記録されることを単体テストで確認します。
- 「完了通知」設定がOFFの時は通知登録されないことを単体テストで確認します。
- 「完了通知」設定がONの時は通知登録されることを単体テストで確認します。
- OS通知バナーと通知音の目視確認は未実施です。

## 4. 今後の予定
バックグラウンド状態で補正またはマスタリングを完了させ、OS通知バナーと通知音が出ることを目視確認します。
