# 完了通知機能 設計書 (Implementation Plan)

## 1. 目的
補正またはマスタリングが完了した時に、macOS標準の通知バナーと通知音で知らせます。

## 2. 実装内容

### `NotificationService.swift`
- アプリ起動時に通知許可を申請します。
- 補正完了時は `補正が完了しました` と通知します。
- マスタリング完了時は `マスタリングが完了しました` と通知します。
- 通知音はmacOS標準音を使います。

### `ProcessingJob.swift`
- 補正完了時に通知サービスを呼びます。
- マスタリング完了時に通知サービスを呼びます。
- 同じ処理の完了通知を重複送信しません。

## 3. テスト
- 補正完了通知が1回だけ記録されることを確認します。
- マスタリング完了通知が1回だけ記録されることを確認します。
- 実機でOS通知バナーと通知音を確認します。
